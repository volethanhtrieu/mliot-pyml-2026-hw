## Quy trình làm và nộp

1. Đồng bộ đề mới từ upstream:

   ```bash
   git pull upstream main
   ```

2. Điền họ tên và MSSV trong notebook.
3. Chỉnh sửa trực tiếp notebook trong đúng thư mục bài tập.
4. Chọn **Restart Kernel** và chạy lại toàn bộ notebook từ trên xuống.
5. Commit và push lên fork cá nhân:

   ```bash
   git add week02/
   git commit -m "week02: hoan thanh bai tap"
   git push origin main
   ```

6. Gửi link fork theo hướng dẫn của mentor.

Không đổi tên notebook, không di chuyển file và không chỉnh sửa dataset được cung cấp.
